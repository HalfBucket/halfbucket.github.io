Windows下Hugo的安装与使用 需要的资源准备（前两项需自行了解） Git的下载与安装、简单git指令的熟悉与使用。 Github账号及其简单的了解。 Hugo安装包下载。 地址：https://github.com/spf13/hugo/releases 安装Hugo 新建Hugo的主文件夹：D/Hugo; 新建Hugo的程序放置位置：D/Hugo/bin; 将下载好的Hugo.zip文件中的三个文件解压至此文件夹，其中包含一个.exe的可执行文件，该文件可以在控制台中直接运行，无需安装。 将 ;D/Hugo/bin添加至系统环境变量的PATH值末尾。 打开命令行键入hugo help,若运行成功则Hugo安装成功。（注意D/Hugo/bin下的.exe文件名若不为hugo，需改成hugo。） 新建网站 在Hugo文件夹下新建 D/Hugo/sites作为我们网站的统一放置路径。 在命令行中执行以下命令新建网站 C:\Users\LH> D: //进入d盘命令 D:\> hugo //进入hugo文件夹 D:\Hugo> cd sites //进入sites文件夹 D:\Hugo\sites> hugo new site example.com //新建example.com的文件夹 若执行成功，则可以看到D:\Hugo\sites\example.com的文件夹。 为网站添加主题 将example.com文件夹初始化为git仓库 D:\Hugo\sites> git init; //将当前文件夹初始化 为git仓库 Initialized empty Git repository in D:\Hugo\sites/.git/ 此处默认已安装好git环境，若未安装请自行了解。
