mysql字符集设置

2020年4月3日
| 阅读

编码格式

1. 程序编码格式设置 
    开发程序编码格式设置，一般在程序右下角可以设置。

2. mysql 图形化化工具及使用cmd的编码格式
    1. 如sqlyog等图形化工具的编码格式设置
    2. cmd命令行的编码格式设置
        cmd的默认格式是gbk；
        使用"chcp 65001"进入utf8编码cmd；
        使用"chcp 936"回到gbk编码cmd；

字符集设置

  1. mysql中字符集

    show variables like '%character%';
    show variables like 'collation%';

    对应的查询结果如下

    +--------------------------   +--------------------------------------+
| Variable_name            | Value                                |
+--------------------------+--------------------------------------+
| character_set_client     | utf8                                 |
| character_set_connection | utf8                                 |
| character_set_database   | utf8                                 |
| character_set_filesystem | binary                               |
| character_set_results    | utf8                                 |
| character_set_server     | utf8                                 |
| character_set_system     | utf8                                 |
| character_sets_dir       | /usr/local/mysql/share/charsets/``

    修改其中所有的"Value"至需要的模式即可。另外可通过修改配置文件的方式修改其中的部分属性。

  2. Url的设置 配置url是加上 ?useSSL=true&useUnicode=true&characterEncoding=UTF-8

    在xml中配置时&要使用转义符&amp;替代.
